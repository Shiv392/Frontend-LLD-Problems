🚀 Frontend LLD Practice Repository (Angular 17+)
Welcome to the Frontend Low-Level Design (LLD) Practice Repository!
This project is focused on building scalable, reusable, and production-ready UI components using modern frontend technologies.

📌 **Purpose**

The main goals of this repository are:
Strengthen Frontend Low-Level Design (LLD) concepts
Build real-world reusable UI components
Prepare for frontend interviews with practical problems
Write clean, scalable, and maintainable code
🛠️ Tech Stack
⚡ Angular 17+
🎨 Tailwind CSS
🧩 PrimeNG
🧠 TypeScript
🧱 Components Covered

This repository includes commonly used UI components that are frequently asked in interviews and used in real applications:

🔽 Multi-Select Dropdown
📂 Dropdown Menu
🔍 Searchable Select
🧾 Modal / Dialog
📊 Data Table (sorting, filtering)
📅 Date Picker
🔔 Toast / Notifications
📁 File Upload UI
🧭 Navigation Components
...and many more coming soon 🚀

🚀 Getting Started
1. Clone the repository
git clone https://github.com/Shiv392/Frontend-LLD-Problems
cd frontend-lld-practice
2. Install dependencies
npm install
3. Run the project
ng serve
Visit 👉 http://localhost:4200/
